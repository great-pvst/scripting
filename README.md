# 👋 Bonjour,

Ici, vous trouverez une collection de **scripts** conçus pour simplifier des tâches répétitives, optimiser des workflows et expérimenter avec l’automatisation système au quotidien.

---

## 🚀 À propos de ce dépôt

Ce dépôt contient des scripts pour automatiser différentes tâches telles que :  

- 💻 Gestion de fichiers et dossiers
- 🌐 Automatisation web (scraping, bots simples)  
- 📊 Traitement de données
- 🔄 Tâches récurrentes du système  
- ⚙️ Intégration avec API externes  

Chaque script est pensé pour être **simple à comprendre et à adapter** si besoin.

---

## 🛠 Technologies utilisées

- Python 🐍  
- Bash / Shell scripting  
- C
